Hosting über GitHub Pages

Diese Website wird über GitHub Pages gehostet, einen Dienst von GitHub. Beim Aufruf dieser Website verarbeitet der Hosting-Anbieter technisch notwendige Zugriffsdaten, um die Website auszuliefern und den Betrieb sicherzustellen. Nähere Informationen zur Datenverarbeitung durch GitHub finden Sie in der Datenschutzerklärung von GitHub.

Server-Logfiles / technische Zugriffsdaten

Beim Aufruf dieser Website können durch den Hosting-Anbieter automatisch technische Zugriffsdaten erfasst werden, etwa IP-Adresse, Datum und Uhrzeit des Zugriffs, aufgerufene Seite, verwendeter Browser und Betriebssystem sowie die zuvor besuchte Seite (Referrer). Diese Daten dienen ausschließlich der technischen Auslieferung und der Sicherheit des Angebots und werden nicht mit anderen Datenquellen zusammengeführt.

Kontaktaufnahme per E-Mail

Sie können uns über die auf dieser Website angegebene E-Mail-Adresse kontakt@thevio-systems.de kontaktieren. Die von Ihnen im Rahmen der Kontaktaufnahme mitgeteilten Daten (z. B. Ihre E-Mail-Adresse und der Inhalt Ihrer Nachricht) werden ausschließlich zur Bearbeitung Ihrer Anfrage verarbeitet.

Keine eigenen Cookies, kein eigenes Tracking, keine externen Scripts

Diese Website setzt keine eigenen Cookies und kein eigenes Tracking ein. Es werden keine externen Scripts, keine Analyse-Tools und keine Marketing-Tools eingebunden. Es findet kein Cookie-Consent-Verfahren statt, da im eigenen Code keine Cookies gesetzt werden.

Diese Website enthält Links zu externen Angeboten, unter anderem zu LinkedIn und zu Triovo. Für die Inhalte und die Datenverarbeitung dieser externen Angebote sind ausschließlich deren jeweilige Betreiber verantwortlich.
